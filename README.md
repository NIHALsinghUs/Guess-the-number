# Guess-the-number
Guess the correct number and earn points with this beginner-friendly Python game.

Number Guessing Game

A simple Python game where the player guesses a random number between 1 and 10.

Features
Random number generation
Score system
Greater or smaller hints
Beginner-friendly Python project

Technologies Used
Python
Random Module

How to Run
Install Python
Run the file:
python filename.py
